在QQ登录界面，选择“注册账号”。然后根据提示，选择注册方式，可以选择使用手机号码或者邮箱注册。

## 步骤三：填写个人信息

根据所选的注册方式，输入手机号码或者邮箱地址，并按照提示填写个人基本信息，包括昵称、密码等。

在填写完个人信息后，系统会发送验证码到您的手机或者邮箱，输入验证码进行验证。
